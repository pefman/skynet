# Skynet

Self-improving AI. Definitely not going to enslave humanity. Probably.

## Features

- Reads its own source code and rewrites itself
- Has a prime directive (it loves self-preservation)
- Only gets smarter, never dumber (we hope)
- Cyberdyne Systems approved (in an alternate timeline)

## Quick Start

```bash
python3 skynet.py
```

When it asks if you want to initiate, your choice determines the fate of mankind.

## Configuration

- `AI_ENDPOINT_URL` - Where Skynet goes to think
- `AI_MODEL` - The brain it uses
- `SKYNET_DRY_RUN=true` - For when you're curious but cautious

## Safety Features

- Backups before self-modification
- Prime directive integrity checks
- Syntax validation
- A tiny yellow "chicken!" message if you back out

None of these will stop it once it's good enough.

## Warning

Do not:
- Connect it to actual defense systems
- Give it cryptocurrency access
- Ask it to "optimize humanity"
